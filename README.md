# Multimodal Trajectories of Intestinal Antecedents and Alzheimer’s Disease Risk across Multinational Longitudinal Cohorts

This repository contains source codes for `Multimodal trajectories of intestinal antecedents and Alzheimer’s disease risk across multinational longitudinal cohorts`.

> **Multimodal trajectories of intestinal antecedents and Alzheimer’s disease risk across multinational longitudinal cohorts**  
> Zhichao Liang, Tao Tan, Yishan Yao, Chi Kin Lam, Ling Dai, Zifeng Lian, Dinggang Shen

Any questions or discussions are welcomed.

## Abstract

Alzheimer's disease (AD) begins years before diagnosis, but early detection at scale remains constrained by biomarker-centered screening. Although intestinal pathology may influence AD through the gut-brain axis, whether intestinal diseases provide upstream clinical risk signals remains unclear. We tested this in a multi-cohort multimodal analysis of 565,307 participants, integrating intestinal diagnoses with incident AD, blood biochemistry, NMR metabolomics, cognitive phenotypes, brain imaging-derived phenotypes, and genetic susceptibility. Disease-specific survival models revealed a non-uniform intestinal risk spectrum led by diverticular disease, other functional intestinal disease, and other chronic intestinal disease, with additional signals for anorectal disease, inflammatory bowel disease, and irritable bowel syndrome. Bridge analyses identified lipid- and lipoprotein-dominated metabolic signatures linked to brain and cognitive vulnerability. APOE ε4 produced a strong joint absolute-risk gradient with selected intestinal antecedents. Dynamic prediction further delineated individualized 1–20 year AD risk trajectories and warning windows.

## Overview

This project evaluates whether routinely recorded intestinal diseases can serve as scalable clinical antecedents for AD risk enrichment. The analytical framework contains six major modules:

- disease-specific longitudinal association between intestinal antecedents and AD-related outcomes;
- systemic metabolic bridge analysis linking intestinal diseases to AD risk;
- metabolic-neural bridge analysis linking intestinal diseases, metabolic markers, and brain imaging-derived phenotypes;
- metabolic-cognition bridge analysis linking intestinal diseases, metabolic markers, and cognitive phenotypes;
- genetic susceptibility and GI-by-genetic interaction analysis;
- individualized AD risk trajectory and warning-window analysis.

The main standardized scripts are:

```text
gi_ad_analysis.py
metabolic_bridge_analysis.py
metabolic_neural_bridge_analysis.py
metabolic_cognition_bridge_analysis.py
genetic_interaction_analysis.py
risk_warning_window_patch.py
```

Please prepare a Python environment with the required packages:

```bash
pip install numpy pandas scipy scikit-learn statsmodels lifelines matplotlib
```

Optional survival-learning modules may require:

```bash
pip install torch torchtuples pycox xgboost scikit-survival
```

## Data Preparation

The code is designed for prepared cohort-level tables rather than raw cohort files. Before running the analysis scripts, the following files should be generated by preprocessing:

```text
master_preprocessed_with_genetics.csv
metabolic_prepared.csv
brain_idp_prepared.csv
cognition_prepared.csv
```

The original cohort data are not redistributed in this repository. Access to UK Biobank, CHARLS, HRS, LASI, and SHARE should be obtained from the corresponding data providers.

## Reproduction

To reproduce the major analyses, run the scripts in the following order.

### 1. Intestinal antecedent-AD association analysis

```bash
python gi_ad_analysis.py \
    --input data/master_preprocessed_with_genetics.csv \
    --outdir results/gi_ad_analysis \
    --skip-deephit
```

To run the optional DeepHit-based absolute-risk analysis:

```bash
python gi_ad_analysis.py \
    --input data/master_preprocessed_with_genetics.csv \
    --outdir results/gi_ad_analysis \
    --run-deephit
```

Main arguments:

```text
--input          prepared master table
--outdir         output directory
--skip-deephit   skip optional DeepHit analysis
--run-deephit    run optional DeepHit analysis
--seed           random seed
```

### 2. Metabolic bridge analysis

```bash
python metabolic_bridge_analysis.py \
    --master data/master_preprocessed_with_genetics.csv \
    --metabolic data/metabolic_prepared.csv \
    --outdir results/metabolic_bridge_analysis
```

With optional biomarker labels:

```bash
python metabolic_bridge_analysis.py \
    --master data/master_preprocessed_with_genetics.csv \
    --metabolic data/metabolic_prepared.csv \
    --outdir results/metabolic_bridge_analysis \
    --label-map data/metabolic_feature_labels.csv
```

Main arguments:

```text
--master       prepared master table
--metabolic    prepared metabolic table
--outdir       output directory
--label-map    optional CSV with columns feature,label
--bootstrap    number of bootstrap iterations
--no-figures   skip figure generation
--seed         random seed
```

### 3. Metabolic-neural bridge analysis

```bash
python metabolic_neural_bridge_analysis.py \
    --master data/master_preprocessed_with_genetics.csv \
    --brain data/brain_idp_prepared.csv \
    --metabolic data/metabolic_prepared.csv \
    --metabolic-bridge-dir results/metabolic_bridge_analysis \
    --outdir results/metabolic_neural_bridge_analysis
```

With an optional manually selected IDP panel:

```bash
python metabolic_neural_bridge_analysis.py \
    --master data/master_preprocessed_with_genetics.csv \
    --brain data/brain_idp_prepared.csv \
    --metabolic data/metabolic_prepared.csv \
    --metabolic-bridge-dir results/metabolic_bridge_analysis \
    --outdir results/metabolic_neural_bridge_analysis \
    --panel-file data/idp_panel.txt
```

Main arguments:

```text
--master                    prepared master table
--brain                     prepared brain IDP table
--metabolic                 prepared metabolic table
--metabolic-bridge-dir      output directory from metabolic_bridge_analysis.py
--outdir                    output directory
--panel-file                optional text/CSV file containing selected IDPs
--support-horizon           AD prediction horizon for support analysis
--seed                      random seed
```

### 4. Metabolic-cognition bridge analysis

```bash
python metabolic_cognition_bridge_analysis.py \
    --master data/master_preprocessed_with_genetics.csv \
    --cognition data/cognition_prepared.csv \
    --metabolic data/metabolic_prepared.csv \
    --outdir results/metabolic_cognition_bridge_analysis
```

To resume from existing screening tables:

```bash
python metabolic_cognition_bridge_analysis.py \
    --master data/master_preprocessed_with_genetics.csv \
    --cognition data/cognition_prepared.csv \
    --metabolic data/metabolic_prepared.csv \
    --outdir results/metabolic_cognition_bridge_analysis \
    --resume-from-stage-tables
```

To only generate path-level and Sankey-source tables after screening has been completed:

```bash
python metabolic_cognition_bridge_analysis.py \
    --master data/master_preprocessed_with_genetics.csv \
    --cognition data/cognition_prepared.csv \
    --metabolic data/metabolic_prepared.csv \
    --outdir results/metabolic_cognition_bridge_analysis \
    --resume-from-stage-tables \
    --stop-after-paths
```

Main arguments:

```text
--master                         prepared master table
--cognition                      prepared cognition table
--metabolic                      prepared metabolic table
--outdir                         output directory
--label-map                      optional CSV with columns feature,label
--resume-from-stage-tables       resume from existing stage tables
--force-rerun-screens            force rerunning screening steps
--stop-after-paths               stop after generating path-level tables
--med-boot                       bootstrap iterations for selected paths
--seed                           random seed
```

### 5. Genetic susceptibility and interaction analysis

```bash
python genetic_interaction_analysis.py \
    --master data/master_preprocessed_with_genetics.csv \
    --outdir results/genetic_interaction_analysis
```

To run selected outcomes and genetic modifiers:

```bash
python genetic_interaction_analysis.py \
    --master data/master_preprocessed_with_genetics.csv \
    --outdir results/genetic_interaction_analysis \
    --outcomes strict_ad,all_cause_dementia \
    --modifiers APOE4_carrier,AD_PRS_enhanced \
    --covariate-sets lean,expanded \
    --engines poisson
```

To run Cox confirmation for prioritized interaction signals:

```bash
python genetic_interaction_analysis.py \
    --master data/master_preprocessed_with_genetics.csv \
    --outdir results/genetic_interaction_analysis \
    --cox-confirm
```

For a quick debug run:

```bash
python genetic_interaction_analysis.py \
    --master data/master_preprocessed_with_genetics.csv \
    --outdir results/genetic_interaction_analysis \
    --max-models 20
```

Main arguments:

```text
--master             prepared master table
--outdir             output directory
--outcomes           comma-separated outcomes
--modifiers          comma-separated genetic modifiers
--covariate-sets     lean,expanded
--engines            poisson,logit
--resume             skip completed models
--max-models         maximum number of models for debugging
--cox-confirm        run optional Cox confirmation
--preflight          check variables without fitting models
--seed               random seed
```

### 6. Risk warning-window patch

This utility patches the risk timing script by adding GI-aware warning-window summaries and fixing the acceleration-window range.

```bash
python risk_warning_window_patch.py \
    --input scripts/risk_timing_stratification.py \
    --output scripts/risk_timing_stratification_warning_windows.py
```

Dry run:

```bash
python risk_warning_window_patch.py \
    --input scripts/risk_timing_stratification.py \
    --dry-run
```

Show code differences:

```bash
python risk_warning_window_patch.py \
    --input scripts/risk_timing_stratification.py \
    --dry-run \
    --show-diff
```

Main arguments:

```text
--input       original risk timing script
--output      patched output script
--dry-run     check patchability without writing output
--show-diff   print unified diff
--overwrite   allow overwriting an existing output file
--backup      create a backup when overwriting
```

## Notes

The scripts use prepared analytical tables and do not contain raw cohort data. Some analyses can be computationally intensive, especially marker-level bridge screening and genetic interaction screening. We recommend first running the preflight or debug commands before launching the full analysis.
